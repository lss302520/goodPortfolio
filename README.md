# [메인화면]
 ![룍](https://user-images.githubusercontent.com/84943154/134843836-3c367802-af1b-44f4-988d-355b42dd01bd.PNG)
### 메인화면 구성으로 동물에 대해 잘 알 수 있게 도움을 주는 컨셉을 잡고 만든 홈페이지로 로그인하는 곳과 회원가입 검색창 등을 넣어 구현했습니다.(검색창 기능을 구현하기에 어려움을 느껴 틀만 만들었습니다.)

# [회원가입]
##![회](https://user-images.githubusercontent.com/84943154/134843656-e0a8b6aa-98d8-4558-bc31-c00cba8804de.PNG)

### 회원가입을 누르면 회원가입 창이 나옵니다.
![회원가입](https://user-images.githubusercontent.com/84943154/134843721-31bc0b6c-6398-497d-b86d-e75ce504fb95.PNG)

### 비밀번호가 일치한지 정보의 정확성을 위해 비밀번호 확인 기능도 구현해봤습니다.
![비번](https://user-images.githubusercontent.com/84943154/134848312-7bf43b05-578f-46f1-a840-f8dd4a7e2d1a.PNG)

### 회원가입 창에 정보를 입력하고 가입을 누릅니다. 가입을 누르면 db연동을 한 테이블로 정보를 넘겨 로그인 할 때 다시 넘겨 받을 수 있도록 하였습니다.
![회원 과정](https://user-images.githubusercontent.com/84943154/134847463-c5c62a41-89cd-4841-b158-b36b497cd09f.png)

### 가입이 끝나면 로그인을 바로 할 수 있도록 로그인 창으로 이동합니다.  
![로그인](https://user-images.githubusercontent.com/84943154/134845630-324f8284-90f0-45dd-9579-afd26695128c.PNG)
